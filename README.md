Уже 15я неделя разработки. В итего игра полностью приобразилась. Поменялась общая концепция игры, теперь на выбор не будет 3 персонажа, а один главный герой с 3 уровнями.
На кажом уровне у персонажа улучшаются характеристики, что как раз нужно для RPG игры.
Добавлены новые текстуры, новая боевка и система уровней.
До финального этапа осталось только сделать босс-файт и мелкие штрихи.
Так же скоро будет реализован звук и OST.



Что готово:

1) Полноценная анимация по спрайтам
2) Рабочее главное меню с START, DEV MODE, EXIT
3) Режим разработчика который переключается с кнопкой DEV MODE
4) Обновленная боевка. Атаки по КД и разные радиусы атак у каждого перса
5) Создать 3 основыне карты (Overworldd, Dungeon, Nether)
6) Доработать новую боевку и добавить индивидууальные скилы.
7) Новые текстуры персонажей и врагов.
8) Создание первых уровней и систему геймплея.
9) Реализовать переход между уровнями.
10) Анимировать атаку для персонажа.
