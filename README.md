За пару недель проект усовершенствовался от сырого до чуть чуть не сырого.
Преждние механики остались такими же но есть пару изменений.
Самое главное это добавление главного меню и полностью переработанная боевка.
Теперь враги не просто исчезают при касании с персонажем а теперь надо прожимать F для атаки.
У каждого персонажа свой стиль боевки, свои механики и скилы (их пока что нет).
В дальнейшем планируется исминение дизайна главного меню. текстур и сущностей.
Так же будет реализована переход на другие карты (но это в будущем).






Что готово:

1) Выбор персонажа при начале игры
2) Полноценная анимация по спрайтам
3) Рабочее главное меню с START, DEV MODE, EXIT
4) Режим разработчика который переключается с кнопкой DEV MODE
5) Обновленная боевка. Атаки по КД и разные радиусы атак у каждого перса

Что планируется:

1) Создать 3 основыне карты (Overworldd, Dungeon, Nether)
2) Доработать новую боевку и добавить индивидууальные скилы.
3) Новые текстуры персонажей и врагов.
4) Создание первых уровней и систему геймплея.
5) Реализовать переход мезду уровнями.
6) Реализовать систему лечения.
7) Реализовать систему пополнения стрел для лучника.
8) Анимировать атаку для кардого персонажа.
