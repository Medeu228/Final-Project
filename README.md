Уже 15я неделя разработки. В итего игра полностью приобразилась. Поменялась общая концепция игры, теперь на выбор не будет 3 персонажа, а один главный герой с 3 уровнями.
На кажом уровне у персонажа улучшаются характеристики, что как раз нужно для RPG игры.
Добавлены новые текстуры, новая боевка и система уровней.
До финального этапа осталось только сделать босс-файт и мелкие штрихи.
Так же скоро будет реализован звук и OST.



Что готово:

1) Выбор персонажа при начале игры
2) Полноценная анимация по спрайтам
3) Рабочее главное меню с START, DEV MODE, EXIT
4) Режим разработчика который переключается с кнопкой DEV MODE
5) Обновленная боевка. Атаки по КД и разные радиусы атак у каждого перса
6) Создать 3 основыне карты (Overworldd, Dungeon, Nether)
7) Доработать новую боевку и добавить индивидууальные скилы.
8) Новые текстуры персонажей и врагов.
9) Создание первых уровней и систему геймплея.
10) Реализовать переход между уровнями.
11) Анимировать атаку для персонажа.
